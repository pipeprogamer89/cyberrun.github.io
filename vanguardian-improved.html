Guía de instalación

Instala VanGuardian en tu equipo

Sigue estos sencillos pasos para tener el sistema listo en pocos minutos.

1

Preparar el instalador

VanGuardian OS utiliza la tecnología penguins eggs. Primero, instala Calamares:

sudo apt install calamares

Si hay errores de dependencias, corrige con:

sudo apt --fix-broken install
2

Configurar Calamares

Ejecuta el siguiente comando para configurar el instalador:

sudo eggs calamares --install
3

Iniciar la instalación

Lanza el instalador gráfico con permisos de administrador:

sudo calamares

Sigue el asistente. Al finalizar, puede aparecer un error visual (no afecta). Simplemente reinicia y VanGuardian estará listo.
